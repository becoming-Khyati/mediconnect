# MediConnect 🏥

A full-stack hospital management system built with React and Django that enables seamless interaction between patients, doctors, and administrators.

---

## 🚀 Features

* 👤 Role-based authentication (JWT)
* 🔐 Secured Doctor Profile Creation only by Admin's approval
* 🧑‍⚕️ Doctor & Patient dashboards
* 📅 Appointment booking system with notification on change of status
* 💊 Digital prescription generation
* 📃 Reports uploaded by patient, accessible by doctor
* 📊 Analytics dashboard
* 🔍 Search and filtering system

---

## 🛠️ Tech Stack

**Frontend:** React, Material UI
**Backend:** Django, Django REST Framework
**Database:** SQLite
**Authentication:** JWT

## ⚙️ Setup Instructions

### Backend

cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend

cd frontend
npm install
npm start
